DataGit Guidelines

This is a very early draft of the website aiming to further facilitate data exchange via use of git-annex by Joey Hess.

Some of the data referred to on this portal might not be available from it, and might even require you contacting respective authors to obtain access to the data or to corresponding encryption key(s).

If you use this repository frequently, consider adding the following snippet to ~/.gitconfig and using the third clone URL listed for each project:

[url "http://datagit.org/"]
  insteadOf = dg:
